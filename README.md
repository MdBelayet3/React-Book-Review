
# React Book Review

This is a simple and responsive book review web application built using React. The main goal of this project is to practice React fundamentals and create a clean user interface where users can explore books and read reviews.

## Live Demo

https://react-book-review375.netlify.app/

## Repository

https://github.com/MdBelayet3/React-Book-Review

## Features

* View a list of books
* See detailed information about each book
* Read reviews in a clean layout
* Simple and user-friendly design
* Responsive for different screen sizes

## Technologies Used

* React.js
* React Router
* Tailwind CSS

## How to Run Locally

Clone the project and install dependencies:

```
git clone https://github.com/MdBelayet3/React-Book-Review.git
cd React-Book-Review
npm install
npm run dev
```

Then open your browser and go to:
http://localhost:5173/
